# Git Workshop

Detta repository innehåller material, övningar och instruktioner för Git Workshop.

## Syfte

Syftet med workshopen är att ge grundläggande kunskaper i Git och GitHub inför kommande projekt och kurser. Efter genomförd workshop ska deltagarna kunna arbeta med versionshantering både individuellt och i grupp.

## Förkunskaper

- Grundläggande datorvana
- Git installerat
- GitHub-konto
- Visual Studio Code

## Innehåll

### Dag 1 – Introduktion till Git och GitHub
- Versionshantering
- Git
- GitHub
- Installation och konfigurering

### Dag 2 – Första commits
- git init
- git status
- git add
- git commit
- git log

### Dag 3 – GitHub och samarbete
- git clone
- git push
- git pull
- git restore
- git revert

### Dag 4 – Brancher och Pull Requests
- git branch
- git checkout
- git switch
- git merge
- Pull Requests
